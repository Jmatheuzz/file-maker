# file-maker
Conversor de arquivos usando Nodejs, React e RabbitMQ.

## Features

### Converter de PDF para:
* JPG
* WORD
* EXCEL
* POWERPOINT
* PDF/A

### Converter para PDF de:
* JPG
* WORD
* EXCEL
* POWERPOINT
* HTML

### Operações em PDF
* JUNTAR
* DIVIDIR
* COMPRIMIR
* INSERIR MARCA D'ÁGUA
* EDITAR
